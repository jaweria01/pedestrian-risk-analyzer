# Context-Aware Pedestrian Risk Analyzer

This project presents a prototype context-aware intervention system that predicts pedestrian distraction risk using simulated mobile sensing features.

## Motivation
Pedestrian safety is increasingly affected by mobile distraction. This project explores how contextual signals such as phone usage, walking speed, head posture, and traffic density can be used to predict unsafe situations and trigger preventive alerts.
